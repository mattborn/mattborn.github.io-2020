Page A

After spending 20+ hours on this experiment, I am deeming it a failure. Here’s what I learned:

  • A page needs more reference for its state than just a route and template. Perhaps, a config object per page?
  • It would probably be better to have a special character for “progressive routes” that extend the current context instead of multiple data attributes. The Vanilla JS Toolkit does this, but their code is sloppy.
  • Since the process of distilling down to the necessary bits of logic was so mind-bending, I should consider making this an open source project.
  • Either way, there are plenty of good bits in this code and I shall try again later.
